This package contains independant validation routines.

Purpose

Commons Validator serves two purposes:

  • To provide standard, independant validation routines/functions.
  • To provide a mini framework for Validation.

This package has been created, since version 1.2.1, in an attempt to clearly separate these two concerns and is the location for the standard, independant validation routines/functions in Commons Validator.

The contents of this package have no dependencies on the framework aspect of Commons Validator and can be used on their own.

Date and Time Validators

The date and time validators either validate according to a specified format or use a standard format for a specified Locale.

  • Date Validator - validates dates converting to a java.util.Date type. Also provides functions to compare dates.
  • Calendar Validator - validates dates converting to a java.util.Calendar type. Also provides functions to compare dates.
  • Time Validator - validates times converting to a java.util.Calendar type. Also provides functions to compare times.

Numeric Validators

The numeric validators either validate according to a specified format or use a standard format for a specified Locale.

  • Byte Validator - validates numbers converting to a java.lang.Byte type. Also provides functions for minimum, maximum and range checks.
  • Short Validator - validates numbers converting to a java.lang.Short type. Also provides functions for minimum, maximum and range checks.
  • Integer Validator - validates numbers converting to a java.lang.Integer type. Also provides functions for minimum, maximum and range checks.
  • Long Validator - validates numbers converting to a java.lang.Long type. Also provides functions for minimum, maximum and range checks.
  • Float Validator - validates numbers converting to a java.lang.Float type. Also provides functions for minimum, maximum and range checks.
  • Double Validator - validates numbers converting to a java.lang.Integer type. Also provides functions for minimum, maximum and range checks.
  • BigInteger Validator - validates numbers converting to a java.math.BigInteger type. Also provides functions for minimum, maximum and range checks.
  • BigDecimal Validator - validates numbers converting to a java.math.BigDecimal type. Also provides functions for minimum, maximum and range checks.
  • Currency Validator - validates numbers in using the currency format for a specified Locale. Validated values are converted to a java.math.BigDecimal type.
  • Percent Validator - validates numbers in using the percentage format for a specified Locale. Validated values are converted to a java.math.BigDecimal type.
